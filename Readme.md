## OPEN MUSIC

Open Music App untuk memperdalam fundamental backend dan belajar docker

### Container:

- Open Music Api
- Notification Api
- Redis
- RabbitMQ
- Mailhog
- PostgreSQL

### How to use Docker:

- copy file .env-example ke .env file

```bash
cp .env-example .env
```

- menggunakan docker compose

```bash
docker compose up -d
```
