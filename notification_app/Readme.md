## Notification App

Notification App untuk handle send email (consume data email dari message broker)

### Tech:

- NodeJS
- RabbitMQ
- Nodemailer

### How to compile:

- copy file .env-example ke .env file

```bash
cp .env-example .env
```

- update environtment sesuai dengan kebutuhan

- start app

```bash
npm run start-dev
```
